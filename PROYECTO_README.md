# Sistema de Gestión de Políticas de Negocio

Proyecto completo para la gestión de políticas de negocio en empresas, permitiendo:

1. **Diseño de políticas de negocio** - Crear flujos de trabajo con arrastrar y soltar
2. **Seguimiento de trámites** - Monitor en tiempo real del estado de solicitudes
3. **Asignación de tareas** - Distribuir trabajo entre funcionarios
4. **Análisis de cuellos de botella** - Identificar ineficiencias con IA
5. **Colaboración** - Diseño colaborativo de políticas entre múltiples usuarios

## Stack Tecnológico

- **Frontend**: Angular 17 (TypeScript)
- **Backend**: Python 3.x con FastAPI
- **Base de Datos**: MongoDB
- **Autenticación**: JWT
