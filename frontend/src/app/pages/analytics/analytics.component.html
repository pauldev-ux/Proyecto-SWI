Reglas del Flujo de Trámite

→
Flujo Secuencial

Las actividades se ejecutan en orden. Una actividad comienza después de que la anterior se completa.

Ejemplo: Recibir Solicitud → Revisar Documentos → Validar Presupuesto

⟲
Flujo Iterativo

Si una validación falla, el proceso regresa a una actividad anterior para correcciones.

Ejemplo: Si faltan documentos, regresa a "Revisar Documentos" desde Legal

◇
Punto de Decisión

Compuerta de validación que determina si continúa el flujo o retorna.

Ejemplo: ¿Documentos Completos? → Sí (continúa) / No (retorna)

🔄
Transferencia Interdepartamental

El trámite se transfiere entre departamentos según el flujo establecido.

Ejemplo: Ventas → Legal → Finanzas → Legal → Finalización
